# Comparing Linked List and List in Big O

|    |      Linked list      |  List |
|----------|:-------------:|------:|
| Append |  O(1) | O(1) |
| Pop |    O(n)   |   O(1) |
| Prepend | O(1) |    O(n) |
| Pop first |  O(1) | O(n) |
| Insert  |    O(n)   |  O(n)  |
| Remove | O(n) |    O(n) |
| Lookup by index  |  O(n) | O(1) |
| Lookup by value  |    O(n)   |   O(n) |
